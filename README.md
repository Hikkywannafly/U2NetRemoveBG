# Ứng dụng Tạo Ảnh Thẻ API

Backend API sử dụng FastAPI cho ứng dụng tạo ảnh thẻ với chức năng xóa phông nền.

## Tính năng

- Upload ảnh
- Xóa phông nền tự động
- Tạo ảnh thẻ với nhiều kích thước chuẩn
- Tùy chỉnh màu nền
